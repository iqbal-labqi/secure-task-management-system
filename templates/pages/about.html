Security

Security Features Implemented

Every control maps to OWASP Top 10 or ASVS requirements.

{% with features='' %}
OWASP Top 10 Mitigations
Addresses A01–A10 including injection, auth failures, misconfig
Role-Based Access Control (RBAC)
Admin and User roles with strict view/data isolation
CSRF Protection
Django CSRF middleware + tokens on all state-changing forms
Server-Side Input Validation
All inputs sanitized with bleach and validated via Django forms
Secure Authentication
PBKDF2 password hashing, 30-min session timeout, secure cookies
Secure File Upload
MIME verification via python-magic, UUID rename, 2MB limit
Audit Logging & Monitoring
Immutable logs for login, task CRUD, admin actions — no passwords logged
Brute Force Protection
django-axes: lockout after 5 failed login attempts (1-hour cooldown)
Security Headers
CSP, X-Frame-Options DENY, HSTS, X-Content-Type-Options
IDOR Prevention
All task queries filtered by owner — prevents horizontal privilege escalation
